Overview

AISidekick provides hosted MCP (Model Context Protocol) services for Oracle Primavera P6 and Primavera Unifier integrations.

Supported AI platforms can securely connect to enterprise Primavera environments using authenticated MCP endpoints.

Authentication

Every AISidekick subscription includes a secure API key for MCP authentication. 


Authorization: Bearer YOUR_API_KEY

MCP Endpoints

Use the following hosted endpoints to connect your AI platforms.

P6 Sidekick Endpoint 

https://api.aisidekick.ai/mcp/p6
Unifier Sidekick Endpoint 

https://api.aisidekick.ai/mcp/unifier

Claude Desktop Integration

Configure Claude Desktop to use AISidekick MCP endpoints. 


{
  "mcpServers": {
    "p6-sidekick": {
      "command": "npx",
      "args": [
        "-y",
        "@modelcontextprotocol/server-fetch",
        "https://api.aisidekick.ai/mcp/p6"
      ],
      "env": {
        "AUTH_TOKEN": "YOUR_API_KEY"
      }
    }
  }
}

Cursor Integration

Cursor supports MCP integrations using hosted endpoints and authenticated API access.

OpenAI Integration

AISidekick MCP infrastructure can be integrated into OpenAI-powered enterprise workflow applications.

API Examples

Example authenticated request using Python. 


import requests

headers = {
    "Authorization": "Bearer YOUR_API_KEY"
}

response = requests.get(
    "https://api.aisidekick.ai/mcp/p6",
    headers=headers
)

print(response.json())

Troubleshooting

Authentication Errors

Verify that your API key is active and included in the Authorization header.

MCP Connection Issues

Ensure the MCP endpoint URL is correct and accessible from your AI client.

Unsupported AI Client

Verify that your AI platform supports MCP integrations or external connectors.
